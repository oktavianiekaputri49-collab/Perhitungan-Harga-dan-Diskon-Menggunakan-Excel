📊 Perhitungan Harga dan Diskon Menggunakan Excel
Deskripsi
Proyek ini berfokus pada pengolahan data harga dan diskon menggunakan Microsoft Excel. Data dianalisis untuk menghitung total harga pembelian berdasarkan jumlah stok, serta potongan diskon dengan persentase tertentu.

Dataset
Data terdiri dari:
- Harga per pcs
- Jumlah stok dibeli
- Harga total
- Diskon (10% dan 20%)

Tools & Fungsi yang Digunakan
- Microsoft Excel
- Rumus perkalian untuk menghitung total harga
- Rumus persentase untuk menghitung diskon
- Referensi sel absolut (F4)

Hasil Analisis
- Total harga pembelian berhasil dihitung
- Diskon 10% dan 20% dapat ditentukan secara otomatis
- Perhitungan menjadi lebih cepat dan akurat

Tujuan Proyek
- Melatih penggunaan rumus dasar di Excel
- Memahami perhitungan diskon
- Mengolah data menjadi informasi yang terstruktur

Insight
Dengan menggunakan rumus Excel, perhitungan harga dan diskon dapat dilakukan secara otomatis sehingga mengurangi kesalahan dan meningkatkan efisiensi.

Penutup
Proyek ini merupakan latihan dasar dalam pengolahan data keuangan sederhana menggunakan Excel.
